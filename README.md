# Story-Verse 

A comprehensive web platform for readers and storytellers to explore, share, and engage with captivating narratives across multiple genres. This project was developed as part of an Introduction to Information and Communication Technologies (IICT) group project.

## Key Features
- User Authentication: Secure login and signup functionality for personalized access.
- Dynamic Home Page: A visually engaging layout that showcases featured stories, popular categories, and the latest additions.
- Organized Content: Well-structured categories like Fiction, Fantasy, Adventure, Science, Comics, Mystery, Poetry, and Biographies.
- Interactive Interface: Responsive design and intuitive navigation.

## Tech Stack
- Frontend: HTML, CSS, JavaScript.
- Tools: Visual Studio Code, Git.

## Getting Started
1. Clone the repository:
   git clone https://github.com/its-4mna/Story-Verse.git
2. Navigate to the project folder:
   cd Story-Verse
3. Open the website: double-click home.html in your file explorer.
